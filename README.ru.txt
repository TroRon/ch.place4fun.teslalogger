У вас установлен TeslaLogger, который записывает данные вашего автомобиля?

Вы хотите иметь возможность просматривать эти данные, лучше всего через Homey?

Тогда у меня есть что-то для вас:
Расширьте Homey, чтобы отображать данные вашего TeslaLogger и реагировать на них.

Возможности бесконечны, когда вы расширяете Homey с этой функцией.

Сначала начните с добавления драйвера "TeslaLogger". Он является основой для всех остальных драйверов.

Приятного использования этого приложения!
