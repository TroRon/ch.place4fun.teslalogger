Sie haben eine TeslaLogger Installation welche die Daten Ihres Wagens mitschreibt?

Sie möchten die Daten einfach einsehen können, am Besten über den Homey?

Dann habe ich hier was für Sie:
Erweitern Sie Homey um die Möglichkeit, Daten ihres TeslaLogger anzuzeigen und darauf zu reagieren.

Die Möglichkeiten sind endlos, wenn Sie Homey mit dieser Funktion erweitern.

Beginnen Sie als Erstes damit, den Treiber "TeslaLogger" hinzuzufügen. Er ist die Basis für alle weiteren Treiber.
